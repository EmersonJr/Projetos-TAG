## Rodando o codígo:

Para rodar o codígo inicialmente é necessário compilar e gerar um executável, isso pode ser feito executando o seguinte comando:

```shell
g++ main.cpp -o main
```
Com o codígo compilado e o executável gerado, execute o programa com:

```shell
./main
```
